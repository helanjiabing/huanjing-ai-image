Questions

使用前，可能想知道

这个网站是完全免费的吗？

网页本身免费使用。免费备用模型可能需要排队；FLUX.1 Schnell、Nano Banana 2 等高质量 API 通常需要账户余额，因此适合作为少量演示线路并设置次数限制。

为什么不同用途生成的构图不一样？

手机与聊天背景需要为时间、图标和聊天文字留白；电脑桌面强调宽幅景深；头像则更重视主体居中和小尺寸辨识度。

生成的图片可以商用吗？

是否能商用取决于所选模型的服务条款、提示词内容及是否涉及第三方版权。正式商业使用前，应再次核对对应模型的最新许可与内容政策。

提示词需要写得很专业吗？

不需要。选择用途与风格后，输入框会自动出现可编辑模板；你只要替换主体、颜色或氛围即可。
